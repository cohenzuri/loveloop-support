# LoveLoop Support Website

Official support website for **LoveLoop** - A relationship activity tracking app for iOS and Apple Watch.

## About LoveLoop

LoveLoop helps couples track and celebrate their relationship activities together. Whether it's hugs, kisses, dates, or quality time, LoveLoop makes it easy to set goals and stay connected.
